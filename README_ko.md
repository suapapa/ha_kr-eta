# KR ETA for Home Assistant

![logo](https://github.com/suapapa/ha_kr-eta/blob/main/icons/logo.png)

`kr_eta`는 Home Assistant의 custom component로 두 주소 사이의 이동 시간을 예측하는 기능을 제공합니다.

## Prerequirements

- [카카오모빌리티 디벨로퍼스 > 문서 > 길찾기API > 시작하기](https://developers.kakaomobility.com/docs/navi-api/start/)의 가이드를 따라 앱 등록 및 REST API 키 확인.
- [국토교통부 - V-WORLD 디지털트윈국토 > 오픈API](https://www.vworld.kr/v4po_openapi_s001.do)에서 Geocoder API 2.0을 사용하기 위한 API 키 발급.
