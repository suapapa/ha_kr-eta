# KakaoNavi ETA for Home Assistant

`ha-kakaonavi` is a Home Assistant custom component that integrates with Kakao Mobility's Navigation API
to provide real-time estimated travel times and routes between specified locations.

## Prerequirements

[카카오모빌리티 디벨로퍼스 > 문서 > 길찾기API > 시작하기](https://developers.kakaomobility.com/docs/navi-api/start/)의
가이드를 따라 앱 등록 및 REST API 키 확인.
