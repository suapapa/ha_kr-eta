# KR ETA for Home Assistant

![logo](https://github.com/suapapa/ha_kr-eta/blob/main/icons/logo.png)

`kr_eta` is a custom component for Home Assistant that provides travel time estimation between two addresses.

## Prerequisites

- Follow the guide at [Kakao Mobility Developers > Docs > Directions API > Getting Started](https://developers.kakaomobility.com/docs/navi-api/start/) to register an application and obtain a REST API key.
- Issue an API key for using Geocoder API 2.0 at [Ministry of Land, Infrastructure and Transport - V-WORLD Digital Twin > Open API](https://www.vworld.kr/v4po_openapi_s001.do).
